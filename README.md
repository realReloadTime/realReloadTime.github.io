# Портфолио на GitHub Pages

Одностраничное портфолио для фриланса.

## Публикация

1. Создайте на GitHub публичный репозиторий **`realReloadTime.github.io`**
2. Замените в `index.html` плейсхолдеры:
   - `YOUR_TELEGRAM_USERNAME`
   - `YOUR_EMAIL@example.com`
   - `YOUR_KWORK_USERNAME`
3. Запушьте:

```bash
cd c:\Users\maxsh\PycharmProjects\realReloadTime.github.io
git init
git add .
git commit -m "Add portfolio landing page"
git branch -M main
git remote add origin https://github.com/realReloadTime/realReloadTime.github.io.git
git push -u origin main
```

4. GitHub → Settings → Pages → Source: **Deploy from branch `main` / root**

Сайт будет доступен: https://realReloadTime.github.io

## Альтернатива

Скопируйте `index.html` в Notion (embed) или Tilda как HTML-блок.
